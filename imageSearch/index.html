Image Search Abstraction Layer

User Stories

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can skip through the responses by adding a ?offset=N parameter to the URL.
  • I can get a list of the most recently submitted search strings.

Image search powered by Getty Images

To search for images of wombats:

https://fcc-projects-szechuan-sage.c9users.io/search/wombats

Sample Output

[{"id":"53735484","uri":"http://cache2.asset-cache.net/xt/53735484.jpg?v=1&g=fs2|0|editorial_asiapac|35|484&s=1&b=NTU=","title":"Spring Baby Boom At Taronga Zoo"},...]


To search for images of wallabies starting from the second page:

https://fcc-projects-szechuan-sage.c9users.io/search/wallabies?offset=2

Sample Output

[{"id":"101291721","uri":"http://cache3.asset-cache.net/xt/101291721.jpg?v=1&g=fs2|0|editorial38|91|721&s=1&b=OA==","title":"Wallabies Berrick Barnes And Drew Mitchell Meet Model Kristy Hinze"},...]


To show the latest searches:

https://fcc-projects-szechuan-sage.c9users.io/latest

Sample Output

[{"query":"wallabies?offset=2&","time":"2016-06-25 11:19:19"},{"query":"wombats","time":"2016-06-25 11:18:56"}]
